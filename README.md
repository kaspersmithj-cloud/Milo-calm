# Milo Trainer Pro v3

Mobile-first iPhone version.

## Replace files in GitHub
Upload these files into the root of your repo and overwrite old files:
- index.html
- styles.css
- app.js
- programme.js
- manifest.json
- sw.js

Then commit changes. GitHub Pages updates automatically.

If iPhone shows the old design, open Safari and refresh twice, or remove the Home Screen app and add it again.
